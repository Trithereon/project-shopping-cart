# Shopping Cart

## Homepage

- Farm theme.
- Header, Main, Footer
- Hero, slogan, about us, contact info, shop button, nav bar.

### Components

1. Nav bar `<Nav>`
2. Hero section `<Hero>` + `<HeroImg>`
3. Featured item cards? Or save them for the Shop page `<FeatItemCard>`
4. Text boxes for slogan and info `<TextBox>`
5. Header `<Header>`
6. Footer `<Footer>`

## Shop Page

- Food items from FakeStoreAPI.

### Components

1. Nav bar `<Nav>`
2. Featured item cards `<FeatItemCard>`
3. Full grid list of items `<GridList>`
4. List item cards `<ItemCard>`
5. Search bar `<SearchBar>`

## Cart Page

- Seems straight-forward.
